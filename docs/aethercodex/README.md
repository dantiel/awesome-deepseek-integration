## 🚀 Getting Started

### Prerequisites

- [TextMate](https://macromates.com/) 2.0+
- Ruby 3.0+
- A DeepSeek API key

### Installation

```bash
git clone https://github.com/dantiel/aethercodex.git
cd aethercodex
cd Support && bundle install
```

Install the TextMate bundle by double-clicking `AetherCodex.tmbundle` or copying to `~/Library/Application Support/TextMate/Bundles/`.

### Configuration

Create a `.aethercodex` file (project, home, or bundle directory):

```yaml
api_key: "your-deepseek-api-key"
model: "deepseek-chat"
reasoning_model: "deepseek-reasoner"
```

Or use environment variables:

```bash
export DEEPSEEK_API_KEY="your-api-key"
```
